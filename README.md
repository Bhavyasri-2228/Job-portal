# Job-portal
A full stack Job Portal web application built using Python, Django, MySQL, REST API, and React. Features include user registration, job listings, job applications, admin dashboard, and responsive UI with HTML, CSS, and JavaScript.
Job Portal Project – Full Explanation (Start to End)

Project Overview This is a full-stack Job Portal web application developed using:

Backend: Python, Django, Django REST Framework

Frontend: React, HTML, CSS, JavaScript

Database: MySQL

It allows job seekers to register, view jobs, and apply, while recruiters can post jobs and manage applications.

Project Architecture css Copy Edit Job_portal_project/ ├── backend/ → Django project (API, models, auth, etc.) ├── frontend/ → React project (UI, job listings, apply form) ├── screenshots/ → All project UI screenshots └── README.md → Project explanation and documentation

Technologies Used **Layer Technologies Used Frontend React, HTML, CSS, JavaScript Backend Django, Django REST Framework Database MySQL Tools Git, GitHub, VS Code

Main Features 👤 User Side (Job Seeker) Register and login

Browse job listings

View job details

Apply to jobs

See application status in dashboard

🧑‍💼 **Recruiter Side Register and login

Post a new job

View applicants for a job

Accept/Reject applications

API Functionality (Django REST Framework)

/api/jobs/ → List all jobs

/api/jobs// → Job details

/api/apply/ → Submit an application

/api/user/dashboard/ → View user’s applied jobs

Frontend UI (React) Built using React components
